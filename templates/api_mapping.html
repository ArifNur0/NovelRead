Mapping API

Dokumentasi ringkas endpoint yang tersedia di aplikasi.

Endpoint

Kembali
GET /
Dashboard daftar novel (hanya user login).
GET /login
Menampilkan halaman login.
POST /login
Login user. Form: username, password.
GET /register
Menampilkan halaman register.
POST /register
Membuat user. Form: username, password.
GET /logout
Logout dan hapus session.
GET /upload
Halaman upload cover & PDF (hanya user login).
POST /upload
Upload novel. Form-data: title, author, cover(file), pdf(file).
POST /dev/create-user
(Dev) Membuat user cepat untuk testing. Form: username, password.
File storage (static)
Cover: /static/covers/
PDF: /static/pdfs/
Catatan: aplikasi menggunakan penyimpanan JSON untuk users & novels. Endpoint yang melibatkan upload akan menyimpan file ke folder `static/covers` dan `static/pdfs`.
